# My Linktree

A small static link hub for GitHub Pages.

## Edit Links

Update the link labels and URLs in `index.html`:

- Business Email
- YouTube
- Threads
- LinkedIn
- Lecture

## Publish With GitHub Pages

1. Push this repository to GitHub.
2. Open repository settings.
3. Go to Pages.
4. Set the source to the main branch and root directory.

The page uses only static files, so no build step is required.
