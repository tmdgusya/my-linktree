# Fonts

## Ycomputer.otf

- Font name: Y콤퓨타체 / Ycomputer
- Source: KT Y Brand official font page
- Download URL: `https://www.yspotlight.co.kr/file/dl/Y_computer_FONT.zip`
- Official page: `https://www.yspotlight.co.kr/brand/font`

The official KT Y Brand page states: "Y콤퓨타체는 누구나 제약 없이 자유롭게 사용하고 재배포 하실 수 있습니다."

Y콤퓨타체 is used here because it is described as a bitmap typeface with a typewriter-inspired 탈네모 serif structure, and it gives the page a real Korean mechanical-text mood instead of a generic monospace fallback.
